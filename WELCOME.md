## Common Commands

| Command | Description |
|---------|-------------|
| `/help` | Show help information |
| `/clear` | Clear conversation |
| `!` | Run a shell command (e.g., `!git status`) |
| `/init` | Initialize project documentation |

## Tips

- Ask Claude to read, edit, or create files in your workspace
- Use `!` prefix to execute shell commands directly
- Project-specific instructions can be added to `CLAUDE.md`
