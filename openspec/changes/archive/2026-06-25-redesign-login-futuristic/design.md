## Context

当前 `login.html` 是一个单文件登录页面，采用 glassmorphism 风格（半透明卡片 + backdrop-filter blur），紫色渐变主题，包含：
- 8 个浮动粒子动画背景
- 渐变光球背景动画
- 网格覆盖层
- 表单（邮箱/密码/记住我/忘记密码）
- 社交登录（Google/Apple）
- localStorage 认证逻辑

需要将其重新设计为「极简未来」风格，同时保持所有功能不变。

## Goals / Non-Goals

**Goals:**
- 实现暗黑背景 + 霓虹点缀的极简未来视觉风格
- 保持所有现有功能完整可用
- 流畅的微交互动效（聚焦、悬停、点击反馈）
- 保持移动端响应式支持

**Non-Goals:**
- 不新增功能或修改认证逻辑
- 不引入新依赖或框架
- 不修改 `register.html` 或其他文件
- 不做复杂的 3D 或 WebGL 效果

## Decisions

### 1. 配色系统

**决策**：采用 CSS 变量定义的暗黑主题，单一霓虹强调色。

```
--bg-primary: #050505      /* 纯黑背景 */
--bg-card: #0a0a0a         /* 卡片背景 */
--bg-input: #0f0f0f        /* 输入框背景 */
--accent: #00d4ff          /* 电光蓝霓虹 */
--accent-dim: rgba(0,212,255,0.15)  /* 霓虹色半透明 */
--text-primary: #ffffff    /* 主文字 */
--text-secondary: #666     /* 次要文字 */
--text-muted: #333         /* 弱化文字 */
--border: #1a1a1a          /* 边框色 */
```

**理由**：纯黑背景提供最佳对比度，单一霓虹色避免视觉杂乱，符合极简原则。

**备选方案**：
- 多色霓虹 → 放弃，过于花哨
- 渐变强调色 → 放弃，不够克制

### 2. 卡片设计

**决策**：无边框卡片，仅保留极细微的阴影分隔。

- 去掉 `backdrop-filter` 和半透明背景
- 使用纯色 `#0a0a0a` 背景
- 1px `#1a1a1a` 边框（或无边框）
- 微妙的 `box-shadow` 提供层次感

**理由**：glassmorphism 在纯黑背景下效果不明显，简洁的纯色卡片更符合极简未来感。

### 3. 输入框样式

**决策**：底线式输入框，聚焦时霓虹光效。

- 去掉传统边框，仅保留底部 1px 线条
- 聚焦时底部线条变为霓虹色并发光
- 图标在聚焦时也变为霓虹色
- placeholder 使用弱灰色

**理由**：底线式输入框更简洁，聚焦光效提供清晰的交互反馈。

### 4. 按钮设计

**决策**：霓虹边框按钮，悬停时发光扩散。

- 透明背景 + 1px 霓虹色边框
- 悬停时边框发光（box-shadow 扩散）
- 点击时轻微缩放反馈
- 文字使用霓虹色

**理由**：边框式按钮比实心按钮更轻量，符合极简风格，发光效果增强科技感。

### 5. 动效规格

**决策**：使用 CSS transitions 和 keyframes，保持流畅克制。

- 页面载入：卡片淡入 + 微上移（0.6s ease-out）
- 输入框聚焦：边框色过渡（0.3s ease）
- 按钮悬停：shadow 扩散（0.3s ease）
- 按钮点击：缩放（0.15s ease）

**理由**：CSS 动画性能好，时长控制在 0.15-0.6s 之间保持流畅不拖沓。

### 6. 背景处理

**决策**：去掉浮动粒子和渐变光球，保留极简网格。

- 去掉 `.particle` 动画
- 去掉 `.bg::before` 和 `.bg::after` 渐变光球
- 保留或增强网格纹理（更细的线条，更低的透明度）
- 可选：添加极微弱的霓虹光晕作为背景点缀

**理由**：粒子和光球过于花哨，极简风格需要干净的背景。

## Risks / Trade-offs

- **[风险] 霓虹色在低端设备上可能渲染不佳** → 使用标准 CSS 属性（box-shadow, color），避免 filter 和 backdrop-filter
- **[风险] 纯黑背景可能导致某些元素难以辨认** → 确保文字对比度至少 4.5:1，使用足够的边框或背景色差
- **[权衡] 去掉 glassmorphism 可能减少视觉吸引力** → 用霓虹光效补偿，更符合目标风格
- **[权衡] 极简风格可能显得单调** → 通过精确的动效和间距控制保持视觉节奏
