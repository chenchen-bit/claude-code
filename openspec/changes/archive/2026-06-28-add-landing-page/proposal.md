## Why

拓展"创客"品牌系列，当前已有登录页和注册页，缺少一个面向访客的品牌落地主页（landing page）。需要视觉上匹配现有暗色科技风设计语言，同时引入全屏视频背景提升品牌调性和第一印象。

## What Changes

- 新建 `landing.html` — 全屏视频背景的 landing page
- 使用 Pexels 免费可商用视频作为 hero 背景（日落河面云霞，60s，1080p）
- 结构：Hero 全屏视频区 + 品牌简介/理念 + 登录跳转
- 视觉风格延续"创客"系列的暗色 + 青色强调色 (#00d4ff)
